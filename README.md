# Vercel Messenger

Простой мессенджер на Python с бэкендом на Vercel.

## Структура проекта

- `client/` - Python клиент (Tkinter GUI)
- `server/` - Vercel сервер (Serverless Functions)

## Быстрый старт

1. **Клиент:**
```bash
cd client
pip install -r requirements.txt
python main.py
